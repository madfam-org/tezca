## Components
This package exports common UI primitives like:
- Button
- Card
- Badge
- Alert
- Input
- Select
- Progress

## Usage
Import components directly from the package:

```tsx
import { Button, Card } from "@tezca/ui";

export default function MyComponent() {
  return (
    <Card>
      <Button variant="default">Click Me</Button>
    </Card>
  );
}
```
